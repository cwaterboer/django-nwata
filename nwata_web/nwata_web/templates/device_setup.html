📋 Setup Instructions

1 Install Python Dependencies

Ensure you have Python 3.8+ installed, then install required packages:

pip install requests pyobjc-framework-Cocoa  # macOS
# OR
pip install requests pywin32  # Windows
2 Configure API Endpoint

Open nwata_min.py and update the API URL on line 15:

DJANGO_API_URL = "{{ api_url }}"
3 Run the Agent

Start the agent and authenticate with your credentials:

python3 nwata_min.py
First Time Setup:
When you run the agent for the first time, it will prompt you for:
  • Email: {{ user.email }}
  • Password: Your account password
  • Device name: Any name (e.g., "macbook-pro", "work-laptop")
4 Verify Connection

Once authenticated, the agent will start tracking your activity. You should see:

[AUTH] Device registered and token saved.
[SIGNAL] START sent, status 201
[AGENT] Started
[LOG] {'window_title': 'VS Code', 'app_name': 'VS Code', ...}
[SYNC] Uploaded 1 logs

Head over to your Dashboard to see your activity!
