1b apple macOS Only: Handle Security Restrictions

When you download the NwataAgent zip, macOS marks it as "from the internet." Follow one of these methods:

Method 1: Right-Click & Open (Easiest)

  1. Extract the nwata-agent.zip file
  2. Right-click on NwataAgent.app and select Open
  3. Click Open in the security dialog
  4. The app will launch and create a tray icon

Method 2: Terminal Command (Advanced)

After extracting, run this in Terminal to remove the quarantine restriction:

xattr -dr com.apple.quarantine /Applications/NwataAgent.app

Then double-click the app to launch it. If you still see a warning, go to System Settings → Privacy & Security and click Open Anyway for NwataAgent, then try again.

ℹ️ Why this happens: macOS Gatekeeper prevents unsigned apps from unknown developers. This is a normal security feature. You're safe—the agent is open-source and you can verify the code on GitHub.
2 Install Python Dependencies

Ensure you have Python 3.8+ installed, then install required packages:

pip install requests pyobjc-framework-Cocoa  # macOS
# OR
pip install requests pywin32  # Windows
3 Configure API Endpoint

Open nwata_min.py and update the API URL on line 15:

DJANGO_API_URL = "{{ api_url }}"
4 Run the Agent

Start the agent and authenticate with your credentials:

python3 nwata_min.py
First Time Setup:
When you run the agent for the first time, it will prompt you for:
  • Email: {{ user.email }}
  • Password: Your account password
  • Device name: Any name (e.g., "macbook-pro", "work-laptop")
5 Verify Connection
