1b 🍎 macOS Only: Handle Security Restrictions

When you download the NwataAgent zip, macOS marks it as "from the internet." Follow one of these methods:

Method 1: Right-Click & Open (Easiest)

  1. Extract the nwata-agent.zip file
  2. Right-click on NwataAgent.app and select Open
  3. Click Open in the security dialog
  4. The app will launch and create a tray icon

Method 2: Terminal Command (Advanced)

After extracting, run this in Terminal to remove the quarantine restriction:

xattr -dr com.apple.quarantine /Applications/NwataAgent.app

Then double-click the app to launch it. If you still see a warning, go to System Settings → Privacy & Security and click Open Anyway for NwataAgent, then try again.

ℹ️ Why this happens: macOS Gatekeeper prevents unsigned apps from unknown developers. This is a normal security feature. You're safe—the agent is open-source and you can verify the code.
